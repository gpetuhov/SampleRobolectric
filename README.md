# SampleRobolectric
Unit testing an Android app with Robolectric

## Requirements

* Android Studio 3.3 Canary 13
* Kotlin 1.3.0-rc-146
* Android Gradle Plugin 3.3.0-alpha13
* Gradle wrapper 4.10.1
* AAPT 2
